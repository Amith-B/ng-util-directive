# Module import

`import { NgDraggableModule } from 'ng-util-directive';`

<br />

## Draggable Directive Usage

```html
<div
  draggable
  [draggableSticky]="true"
  draggableStickyMargins="1rem"
  draggableStickyTransition="left 0.3s ease"
  draggableTeleportTo="modal-container"
>
  Any content
</div>
```
