# NgContainerResize Usage

`import { NgDebounceEventModule } from 'ng-util-directive';`

<br />

## Handle debounce event

```html
<button
  (debounceEvent)="handleEvent($event)"
  debounceEventName="click"
  [debounceEventTimer]="500"
>
  Debounce Event
</button>

<input
  (debounceEvent)="handleEvent($event)"
  debounceEventName="input"
  [debounceEventTimer]="500"
  placeholder="Type something"
/>
```
