# NgContainerIntersection Usage

## Handle intersection event

```html
<div
  (containerIntersection)="handleContainerIntersection($event)"
  [containerIntersectionOptions]="{rootMargin: '200px', threshold: 1, root: rootEleRef}"
>
  Some Content
</div>
```
